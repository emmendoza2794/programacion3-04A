## Contenido del curso

### Unidad 1 — Elementos del lenguaje C#
*(Semanas 1 a 6)*
- Elementos sintácticos de C#
- Aplicaciones en Consola
- Repaso de Programación Orientada a Objetos (POO)
- Clases, Enumeraciones e Interfaces
- Clase `DateTime` y Clase `Directory`
- Manejo de Archivos de texto
- Manejo de cadenas

### Unidad 2 — Aplicaciones con Interfaz Gráfica de Usuario (GUI)
*(Semanas 7 a 11)*
- BCL (Base Class Library) y Objetos comunes
- Aplicaciones GUI: Objetos Contenedores
- Menús y barras de herramientas
- Creación dinámica de controles
- Librerías de objetos dinámicos

### Unidad 3 — Aplicaciones GUI con MDI y Acceso Desconectado a Datos
*(Semanas 12 a 15)*
- Cache de Datos
- Arquitectura desconectada: `DataSet`, `DataColumn`, `DataRow`, `DataGridView`
- `DataBinding` y `BindingNavigator`
- Aplicaciones con Hilos (`Thread`, `Async`, `Await`)
