Ejercicio único: Sistema de Inventario — "Tienda StoreMax"  (5.0 Puntos)

Objetivo: Desarrollar un formulario de escritorio en C# (Windows Forms, .NET Framework) para gestionar el inventario de productos de una tienda. La aplicación debe guardar y recuperar datos desde un archivo productos.csv.

¿Qué debes hacer?

  1. Registro: Captura Nombre, Categoría, Marca, Precio unitario, Stock disponible, Proveedor y Fecha de ingreso. El ID se genera automáticamente (PRD-001, PRD-002…).
  2. Guardar: Valida que Nombre, Categoría y Precio no estén vacíos; luego agrega el producto a la tabla y lo escribe en productos.csv.
  3. Visualización: La tabla muestra todos los productos. Al seleccionar una fila, sus datos se cargan en el formulario.
  4. Actualizar / Eliminar: Permite editar el producto seleccionado o eliminarlo (con confirmación previa). En ambos casos el CSV se reescribe completo.
  5. Carga inicial: Al abrir la app se leen los registros de productos.csv. Si no existe, inicia sin error.
